# assets/images/

Place your images here.

## Recommended structure

```
images/
├── profile.jpg              ← Your headshot (used on About and Home pages)
├── projects/
│   ├── project-thumb.jpg    ← 800×450 px thumbnail for cards
│   ├── project-hero.jpg     ← 1400×600 px banner for detail pages
│   └── project-1.jpg        ← gallery images
└── posts/
    └── post-cover.jpg       ← blog post cover images
```

## Size recommendations

| Use              | Width  | Height | Format |
|------------------|--------|--------|--------|
| Profile photo    | 600px  | 800px  | JPG    |
| Project thumbnail| 800px  | 450px  | JPG    |
| Project hero     | 1400px | 600px  | JPG    |
| Gallery image    | 800px  | 600px  | JPG    |
| Post cover       | 1200px | 630px  | JPG    |

Compress images before uploading — aim for under 200KB per image.
Free tool: https://squoosh.app
